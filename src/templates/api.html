    • 用户注册

      接口地址：/user/register

      请求方式：POST

      请求参数:空
      参数名 说明 是否必填
      account 用户注册号（手机号码） 是
      password 用户密码 是
      pic 用户头像（转为base64格式） 是
      nickname 用户昵称 是
      sex 用户性别(0=男；1=女) 是
      province 所在省 是
      city 所在市 是

      返回格式：JSON

      返回列表
      code http状态码，只有200表示结果正确
      data 返回的导航数组列表
        -id 导航id号
        -name 导航显示名称
      msg 服务器返回消息

      返回示例

                                  {
                                  "code": 200,
                                  "data": [
                                        {
                                              "id": 101,
                                              "name": "推荐"
                                        },
                                        {
                                              "id": 102,
                                              "name": "母婴"
                                        },
                                        {
                                              "id": 103,
                                              "name": "数码"
                                        },
                                        {
                                              "id": 106,
                                              "name": "test"
                                        }
                                  ],
                                  "msg": "ok"
                            }
